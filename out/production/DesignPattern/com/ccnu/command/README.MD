## 命令模式

将一个对象的请求进行封装成为一个对象， 然后我们可以针对于不同的内容参数进行实例化，对请求排队或者是记录日志的功能 支持撤销的操作的。

### 类的角色

* Command :命令
* Receiver:接受者
* Client:客户端
* Invoker:通过这个调用命令
